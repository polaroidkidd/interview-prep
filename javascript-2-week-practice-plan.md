# JavaScript 2-Week Practice Plan

This plan is optimized for medium-level interview prep using JavaScript.

## Rules For Every Session

1. Spend 2 minutes identifying the pattern before coding.
2. State brute force first.
3. Code without looking at the answer.
4. Test with one normal case and one edge case.
5. If you miss a problem, rewrite the template from memory after reviewing it.

## Week 1: Core High-Yield Patterns

### Day 1: Hash Map / Set

- Review `Map` and `Set` syntax in JavaScript.
- Solve `Group Anagrams`
- Solve `Top K Frequent Elements`
- Solve `Longest Consecutive Sequence`

### Day 2: Two Pointers

- Solve `Container With Most Water`
- Solve `3Sum`
- Solve `Two Sum II`

### Day 3: Sliding Window

- Solve `Longest Substring Without Repeating Characters`
- Solve `Longest Repeating Character Replacement`
- Solve `Permutation in String`

### Day 4: Binary Search

- Solve `Find Minimum in Rotated Sorted Array`
- Solve `Search in Rotated Sorted Array`
- Solve `Koko Eating Bananas`

### Day 5: BFS / DFS

- Solve `Number of Islands`
- Solve `Clone Graph`
- Solve `Pacific Atlantic Water Flow`

### Day 6: Review Day

- Re-solve one missed problem from days 1-5.
- Write the JS template for hash map, two pointers, sliding window, binary search, and DFS from memory.
- Do one timed 30-minute medium.

### Day 7: Mock Day

- Do 2 timed mediums back to back.
- Spend 15 minutes explaining your solutions out loud.

## Week 2: Expansion Patterns

### Day 8: Backtracking

- Solve `Combination Sum`
- Solve `Permutations`
- Solve `Palindrome Partitioning`

### Day 9: 1D DP

- Solve `Coin Change`
- Solve `House Robber II`
- Solve `Decode Ways`

### Day 10: Stack / Monotonic Stack

- Solve `Daily Temperatures`
- Solve `Car Fleet`
- Solve `Evaluate Reverse Polish Notation`

### Day 11: Heap / Priority Queue

- Review the JS `Heap` helper.
- Solve `Kth Largest Element in an Array`
- Solve `K Closest Points to Origin`
- Solve `Task Scheduler`

### Day 12: Prefix Sum

- Solve `Product of Array Except Self`
- Solve `Subarray Sum Equals K`
- Solve `Contiguous Array`

### Day 13: Mixed Practice

- Do 3 mixed mediums without looking at notes.
- After each one, write the pattern name and the core invariant in one sentence.

### Day 14: Final Simulation

- Do 2 full mock interview rounds.
- For each missed step, note whether the issue was:
  - pattern recognition
  - JavaScript syntax
  - edge-case handling
  - testing

## Final Checklist

- Can you write `Map`, `Set`, DFS, BFS, sliding window, binary search, and DP skeletons from memory?
- Can you explain why a pattern fits before you start coding?
- Can you test your code without relying on the platform?
- Can you keep your JavaScript code clean and short under time pressure?
