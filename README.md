## Suggested Usage By Timeline

### If You Have 2 Weeks

1. Read `javascript-interview-prep-handbook.md`
2. Follow `javascript-2-week-practice-plan.md`
3. Use `neetcode-medium-exam-prep.md` during practice
4. Review `javascript-neetcode-templates.md` every few days

### If You Have 2-3 Days

1. Read `javascript-interview-prep-handbook.md`
2. Focus on the high-yield sections in `neetcode-medium-exam-prep.md`
3. Memorize `javascript-neetcode-templates.md`
4. End with `interview-prep-cheat-sheet.md`

### If Your Interview Is Tomorrow

1. Read `day-before-interview-cram-sheet.md`
2. Review `javascript-interview-prep-ultra-compact.md`
3. Skim `javascript-neetcode-templates.md`

## Core Priority Order

If time is limited, study these patterns first:

1. Hash map / set
2. Two pointers
3. Sliding window
4. BFS / DFS
5. Binary search
6. Backtracking
7. 1D DP
